# AnalogClockDemo

クロノグラフアナログ時計

## 必要なライブラリ

- M5Unified

## Arduino IDE 設定

- 開くファイル: `AnalogClockDemo.ino`
- ボード: `m5stack:esp32:m5stack_cores3`

## 操作

1. 画面左上タップ: ストップウォッチ開始 / 再開
2. 画面右上タップ: ストップウォッチ停止
3. 画面左下タップ: スプリット表示の固定 / 解除
4. 画面右下タップ: 停止中のストップウォッチをリセット
5. 電源ボタン: 右側テキスト付き表示と、時計を中央表示するモードを切り替え

## メモ

- メインダイヤルには**時針 / 分針**と、クロノグラフの**センター秒針**を重ねて表示します
- 上サブダイヤルは 1/10 秒・1/100 秒、左サブダイヤルは時計の秒、下サブダイヤルは 0〜5 分の経過を表示します
- ボード側で有効な時刻が取得できる場合はそれを使い、未設定時は**ビルド時刻**を初期値として進めます
- 画面には 4 分割タップ領域のガイドを重ねて表示し、各象限に `START / STOP / SPLIT / RESET` を割り当てています
- 電源ボタンを押すと、**情報パネル表示モード**と**中央時計モード**を切り替えます。どちらのモードでも四隅タップの操作は同じです
